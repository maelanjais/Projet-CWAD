# ADREANALINA PARK

## 📋 Description du Projet

Site web pour un parc sportif et de divertissement développé dans le cadre du projet S106. Ce projet consiste à créer un site internet complet pour un lieu de divertissement avec une identité visuelle cohérente et une expérience utilisateur optimisée.

## 👥 Équipe de Développement

- **JAHIER Maelan**
- **STINUS Rémy**
- **Groupe** : 1M
- **Institution** : IUT d'Orsay
- **Année académique** : 2024-2025

## 🎨 Identité Visuelle

### Palette de Couleurs
Le site utilise une palette basée sur des nuances de vert :

- **#b0ceb0** - Utilisée pour l'entête du site sur toutes les pages
- **#45b645** - Utilisée pour la navigation, le pied de page et les boutons d'accès aux différentes pages

### Typographie
Deux polices principales sont utilisées pour différencier les sections :

- **Geist** - Pour tous les titres (h1 à h6)
- **Vercetti** - Pour le reste du contenu du site

## 🌐 Compatibilité Navigateurs

Le site est optimisé pour :
- ✅ **Chrome** (compatibilité complète)
- ✅ **Microsoft Edge** (compatibilité complète)
- ⚠️ **Firefox** (certains effets visuels peuvent ne pas s'afficher, notamment l'effet de déroulement)

*Note : Les problèmes de compatibilité sont purement esthétiques et n'affectent ni le fonctionnement ni la lisibilité du site.*

## 🚀 Installation et Utilisation

1. **Cloner le repository**
   ```bash
   git clone [https://github.com/maelanjais/Projet-CWAD
   cd adreanalina-park
   ```

2. **Ouvrir le site**
   - Ouvrir `index.html` dans un navigateur web
   - Recommandé : Chrome ou Microsoft Edge pour une expérience optimale

3. **Navigation**
   - Page d'accueil avec présentation du parc
   - Section activités détaillant les services
   - Formulaires de contact et réservation
   - Pages légales (conditions d'utilisation, politique de confidentialité)
